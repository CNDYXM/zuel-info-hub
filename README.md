# 晓观潮 · ZUEL 信息聚合

> 钟声渡晓南 · 七日一巡来

中南财经政法大学信息聚合系统 —— 自动抓取 **24 个校级/院级官网**的通知公告与新闻，仅保留近 **7 日内**内容，按 **9 大智能分类**展示，提供文雅界面与深色模式。

- **版本**：v1.1.1
- **作者**：江渡秋（CNDYXM）  
- **在线地址**：http://118.178.132.19/ （托管于阿里云）  
- **许可证**：MIT
- **立项日期**：2026/6/11

---

## ✨ 特性

- **24 站聚合** — 校级核心 + 16 学院 + 4 职能部门，信息无遗漏  
- **9 类智能分类** — 综合新闻、通知公告、教科动态、讲座预告、师生远方、党建工作、规章制度、就业招聘、志愿服务  
- **潮汐等级（巡潮）** — 根据 7 日内信息总量自动评定 **小潮 / 中潮 / 大潮 / 巨潮 / 风暴潮**  
- **文雅界面** — 宣纸配色、Noto Serif SC 字体、右侧“概览”框含农历节气与巡潮等级  
- **深色模式** — 手动/自动（晚 17 点至早 6 点）切换，偏好持久化  
- **二维筛选** — 按分类 + 来源站点联合筛选，支持标题搜索  
- **全自动巡更** — 点击“巡览”按钮触发全量抓取，自动清理超 7 日旧文

## 🖥 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Python 3.13 + Flask |
| 爬虫 | Crawl4AI（基于 Playwright / Chromium） |
| 前端 | HTML5 + CSS3（CSS 变量 + 深色模式）+ 原生 JavaScript |
| 部署 | Docker + 阿里云 |
| 数据存储 | JSON 文件（单文件缓存，无数据库） |

## 🚀 快速开始

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/CNDYXM/zuel-info-hub.git
cd zuel-info-hub

# 安装依赖
pip install -r requirements.txt

# 下载 Chromium（爬虫必需）
crawl4ai-setup

# 启动服务
python app.py
```

访问 `http://118.178.132.19/` 即可。

> Windows 用户也可双击 `start.bat`（自动创建虚拟环境并启动）。

### Docker 部署

```bash
docker build -t xiaoguanchao .
docker run -p 5000:5000 xiaoguanchao
```

### 阿里云部署（已配置）

项目已打包为 Docker 镜像，部署于阿里云 ECS 或容器服务，访问地址：[http://118.178.132.19/](http://118.178.132.19/)
